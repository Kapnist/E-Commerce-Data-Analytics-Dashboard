# 📊 E-Commerce Data Analytics Dashboard

Dashboard interactif conçu pour transformer des données de vente brutes (+50 000 lignes) en indicateurs de performance visuels.

## 🌟 Points Forts
- **Traitement de données massives** : Utilisation de Python pour transformer un CSV brut en données optimisées.
- **Filtrage Dynamique** : Système de sélection par État (USA) avec mise à jour instantanée des graphiques.
- **Interface Moderne** : Développé avec React 19 et Vite pour une performance maximale.

## 🛠️ Stack Technique
- **Logiciel** : React, Recharts (DataViz).
- **Analyse de données** : Python, Pandas.
- **Performance** : Pipeline de données automatisé (ETL).

## ⚙️ Installation
1. `npm install` pour les dépendances.
2. `python prepare_data.py` pour traiter les 50 000 lignes.
3. `npm run dev` pour lancer l'interface.
